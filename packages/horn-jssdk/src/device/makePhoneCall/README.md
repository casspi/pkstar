
## `jssdk.makePhoneCall`

> `jssdk.makePhoneCall(Object object)`
<br/>
> 版本无要求

### 功能描述

拨打电话。

### 参数

#### `Object object`

| 属性 | 类型 | 默认值 | 必填 | 说明 |
| :--: | :--: | :--: | :--: | :--: |
| phoneNumber | String |  | 是 | 需要拨打的电话号码 |

### 示例代码

```js
jssdk.makePhoneCall({
    phoneNumber: '13111111111'
})
```
