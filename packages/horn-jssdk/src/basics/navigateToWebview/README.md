## `jssdk.navigateToWebview`

> `jssdk.navigateToWebview(Object object)` <br/> 版本无要求

### 功能描述

已一个普通的 webview 打开新 H5 页面。

### 参数

#### `Object object`

| 属性 |  类型  | 默认值 | 必填 |   说明   |
| :--: | :----: | :----: | :--: | :------: |
| url  | String |        |  否  | url 地址 |

### 示例代码

```js
jssdk.navigateToWebview({
  url: 'https://www.baidu.com',
})
```
