
## `jssdk.navigateTo`

> `jssdk.navigateTo(Object object)`
<br/>
> 版本无要求

### 功能描述

打开新H5页面。

### 参数

#### `Object object`

| 属性 | 类型 | 默认值 | 必填 | 说明 |
| :--: | :--: | :--: | :--: | :--: |
| url | String |  | 否 | url地址 |


### 示例代码

```js
jssdk.navigateTo({ 
    url: 'https://www.baidu.com' 
})
```
