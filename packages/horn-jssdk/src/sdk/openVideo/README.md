
## `jssdk.openVideo`

> `jssdk.openVideo(Object object)`
<br/>
> 版本无要求

### 功能描述
播放录像。

### 参数

#### `Object object`

| 属性 | 类型 | 默认值 | 必填 | 说明 |
| :--: | :--: | :--: | :--: | :--: |
| index | Number |  | 是 | - |
| videoInfoList | Array |  | 是 | - |


### 示例代码

```js
jssdk.openVideo({
  // ...
  index: 1,
  videoInfoList:[]
})
```
