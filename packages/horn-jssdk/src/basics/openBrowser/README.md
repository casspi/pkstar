
## `jssdk.openBrowser`

> `jssdk.openBrowser(Object object)`
<br/>
> 版本无要求

### 功能描述

跳转到系统浏览器。

### 参数

#### `Object object`

| 属性 | 类型 | 默认值 | 必填 | 说明 |
| :--: | :--: | :--: | :--: | :--: |
| url | String |  | 是 | 页面地址 |


### 示例代码

```js
jssdk.openBrowser({
    url: `https://www.autostreets.com/`,
})
```
