
## `jssdk.openLivePlayer`

> `jssdk.openLivePlayer(Object object)`
<br/>
> 版本无要求

### 功能描述

直播测试

### 参数

#### `Object object`

|   属性    | 类型 | 默认值 | 必填 | 说明 |
|:-------:| :--: | :--: | :--: | :--: |
| rtmpUrl | String |  | 是 | - |

### 示例代码

```js
jssdk.openLivePlayer({
  // ...
  rtmpUrl: 'xxxx'
})
```
